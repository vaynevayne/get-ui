### 开发步骤

安装依赖: `pnpm install --hoist` , --hoist 是必须的
启动项目: `pnpm sb`, 会启动 storybook,不需要预先 build 项目

组件文档: 后期使用 [nextra](https://nextra.site/) + [sandpack](https://sandpack.codesandbox.io/)

node 版本管理 替换为 volta, 就不需要每次都执行 nvm use
